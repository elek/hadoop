Hadoop Project Mailing Lists

  • User Mailing Lists
  • Project Discussion/Annoucement Mailing Lists
  • Developer Mailing Lists
  • Developer Tracking Mailing Lists

There are many mailing lists in the Hadoop project. Please use the appropriate list described below. In most cases it is necessary to first subscribe to a list before posting to it.

User Mailing Lists

Lists @hadoop.apache.orgPurpose
  common-user@  Primary list for Hadoop user questions.
  hdfs-user@  User questions related specifically to Hadoop HDFS.
  mapreduce-user@  User questions related specifically to Hadoop MapReduce.
  security@  A private list for security sensitive issues. You may post potential security vunerabilities here.

Project Discussion/Annoucement Mailing Lists

Lists @hadoop.apache.orgPurpose
  general@  Hadoop project level annoucements and discussions. All Hadoop releases are announced on this list.
  NOTE: This mailing list is not for end-user questions and discussion. Please use a more-specific subproject's -user mailing list for such issues.

Developer Mailing Lists

Lists @hadoop.apache.orgPurpose
  common-dev@  For those contributing to the development (code, documentation and tests) of the Hadoop Common project.
  hdfs-dev@  For those contributing to the development (code, documentation and tests) of the Hadoop HDFS project.
  mapreduce-dev@  For those contributing to the development (code, documentation and tests) of the Hadoop MapReduce project

Developer Tracking Mailing Lists

Lists @hadoop.apache.orgPurpose
  common-issues@  View all Jira issue changes made in the Hadoop Common Jira. No direct posting allowed.
  common-commits@  View all code changes (diffs) made in the Hadoop Common project. No direct posting allowed.
  hdfs-issues@  View all Jira issue changes made in the Hadoop HDFS Jira. No direct posting allowed.
  hdfs-commits@  View all code changes (diffs) made in the Hadoop HDFS project. No direct posting allowed.
  mapreduce-issues@  View all Jira issue changes made in the Hadoop MapReduce Jira. No direct posting allowed.
  mapreduce-commits@  View all code changes (diffs) made in the Hadoop MapReduce project. No direct posting allowed.
